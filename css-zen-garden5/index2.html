CSS Zen Garden

The Beauty of CSS Design

Uma demonstração do que pode ser alcançado com puro design CSS.

Por favor, carregue o CSS para começar a visualizar diferentes temas.

O Que É Isto?

A ideia do CSS Zen Garden é mostrar como a mesma estrutura HTML pode ter visuais totalmente diferentes apenas trocando a folha de estilo.

Como Funciona

Você mantém este HTML, mas cria seu próprio arquivo style.css com uma nova proposta visual, explorando ao máximo o poder do CSS.

Abreviações Comuns

Termos como CSS e HTML serão vistos ao longo do projeto.
