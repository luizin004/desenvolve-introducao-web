O poder do CSS

Introdução

Este projeto demonstra como um mesmo HTML pode ter visuais completamente diferentes apenas com CSS.

Sobre o Tema

Inspirado na beleza do céu noturno, a paleta traz tons escuros e um toque dourado de estrelas.

Participe

Você pode criar sua própria versão visual apenas editando o arquivo style.css sem modificar este HTML.
